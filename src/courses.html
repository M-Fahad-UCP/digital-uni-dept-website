Courses

A curated list of representative courses. Check the academic catalog for the full curriculum and prerequisites.

Core Courses Electives Projects

CS101 — Introduction to Programming

Foundations of programming using Python. Topics: control flow, functions, and basic data structures.

Offered: Fall, Spring • Credits: 3

Learning Outcomes
  • Write structured programs and functions
  • Apply basic data structures to solve problems
  • Debug and test simple applications

CS201 — Data Structures & Algorithms

Abstract data types, algorithm design, complexity analysis, trees and graphs.

Offered: Spring • Credits: 4

Syllabus Highlights
  • Arrays, linked lists, stacks, queues
  • Trees, heaps, hashing
  • Sorting, searching, complexity

CS310 — Operating Systems

Processes, threads, memory, scheduling, and file systems with practical labs.

Offered: Fall • Credits: 4

Course Projects
  • Process scheduler simulation
  • File system implementation

CS450 — Machine Learning

Supervised/unsupervised learning, model evaluation, and hands-on projects.

Offered: Fall • Credits: 3

Assessment
  • Programming assignments (40%)
  • Project (40%)
  • Final exam (20%)

Signature Projects & Capstones

  • Capstone: Industry-sponsored software product (team of 4–6)
  • Research Project: Semester-long research under faculty supervision
